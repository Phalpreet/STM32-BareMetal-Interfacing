# STM32 Bare-Metal Interfacing Portfolio

Welcome to my embedded systems portfolio. This repository contains my bare-metal C drivers, custom libraries, and hardware interfacing solutions developed for the **STM32G031** microcontroller family. 

Rather than relying heavily on Hardware Abstraction Layers (HAL), these projects demonstrate my ability to read datasheets, manage registers directly, and troubleshoot timing and communication protocols at the lowest level.

| :--- | :--- | :--- |
